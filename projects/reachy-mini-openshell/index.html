Reachy Mini App

Reachy OpenShell

A conversation app for Reachy Mini robot.

Getting Started

Installation

Install this app on your Reachy Mini using the app store, or run it locally:

uv sync
reachy-mini-daemon --sim --scene minimal --headless --no-media
python -m reachy_mini_conversation_app --gradio --no-camera

Configuration

Provider Configuration

Configure provider credentials, model routing, and input mode in .env, then run ./scripts/start-local.sh.
